# Web App - CineMatch (Next.js)

Aplikasi web untuk pemesanan tiket bioskop yang menyediakan fitur pemilihan film, jadwal tayang, pembelian tiket, serta pengelolaan akun pengguna.

## Fungsi Utama
1. Menampilkan daftar film yang sedang tayang
2. Memilih tanggal dan jam tayang
3. Memilih bioskop dan harga tiket
4. Melakukan pembelian tiket
5. Menampilkan riwayat tiket
6. Mengelola profil pengguna
7. Menampilkan promo dan penawaran

## Tech Stack
1. Next.js (App Router)
2. React.js
3. TypeScript
4. Tailwind CSS

## Setup Project
1. Clone repository
2. Masuk ke folder project
3. Install dependencies
4. Jalankan project
5. Buka di browser
